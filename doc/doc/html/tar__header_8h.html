void print_header ( tar_header *  header)

Imprime les informations de l'entrée tar.

Paramètres
headerPointeur vers la structure tar_header qui contient les informations à imprimer.

Cette fonction imprime le nom, la taille, la date de modification et le type de l'entrée tar pointée par header. Si le nom est vide (null-terminated string), la fonction ne fait rien.
